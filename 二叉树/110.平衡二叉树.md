
```
给定一个二叉树，判断它是否是高度平衡的二叉树。

本题中，一棵高度平衡二叉树定义为：

一个二叉树每个节点 的左右两个子树的高度差的绝对值不超过1。

示例 1:

给定二叉树 [3,9,20,null,null,15,7]

    3
   / \
  9  20
    /  \
   15   7
返回 true 。

示例 2:

给定二叉树 [1,2,2,3,3,null,null,4,4]

       1
      / \
     2   2
    / \
   3   3
  / \
 4   4
返回 false 。
```

```
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Depth {
    int value;
    Depth(int value) {
        this.value = value;
    }
}
class Solution {
    public boolean isBalanced(TreeNode root) {
        return dfs(root, new Depth(0));    
    }

    private boolean dfs(TreeNode root, Depth depth) {
        if (root == null) {
            return true;
        }

        Depth leftDepth = new Depth(0);
        Depth rightDepth = new Depth(0);

        if (dfs(root.left, leftDepth) && dfs(root.right, rightDepth)) {
            if (Math.abs(leftDepth.value - rightDepth.value) <= 1) {
                depth.value = Math.max(leftDepth.value, rightDepth.value) + 1;
                return true;
            }
        }

        return false;
    }
}
```
