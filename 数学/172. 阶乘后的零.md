
```
给定一个整数 n，返回 n! 结果尾数中零的数量。

示例 1:

输入: 3
输出: 0
解释: 3! = 6, 尾数中没有零。
示例 2:

输入: 5
输出: 1
解释: 5! = 120, 尾数中有 1 个零.
说明: 你算法的时间复杂度应为 O(log n) 。
```

```
class Solution {
    public int trailingZeroes(int n) {
        int result = 0;
        // n!的0都只有5和2组合产生 比如 10 = 5 * 2 
        // 5 * 4 = 5 * 2 * 2 * 3 * 2 * 1 本质上还是拆分称 5 * 2, 而决定的0个个数可以看出是5
        // 5! = 5 * 4 * 3 * 2 * 1;

        while (n > 0) {
            result += n / 5;
            n /= 5;
        }
        return result;

    }
}
```
