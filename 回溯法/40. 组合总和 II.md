
```
给定一个数组 candidates 和一个目标数 target ，找出 candidates 中所有可以使数字和为 target 的组合。

candidates 中的每个数字在每个组合中只能使用一次。

说明：

所有数字（包括目标数）都是正整数。
解集不能包含重复的组合。 
示例 1:

输入: candidates = [10,1,2,7,6,1,5], target = 8,
所求解集为:
[
  [1, 7],
  [1, 2, 5],
  [2, 6],
  [1, 1, 6]
]
示例 2:

输入: candidates = [2,5,2,1,2], target = 5,
所求解集为:
[
  [1,2,2],
  [5]
]
```

```
class Solution {
    public List<List<Integer>> combinationSum2(int[] candidates, int target) {
        
        List<List<Integer>> result = new ArrayList<>();
        if (candidates == null || candidates.length == 0) {
            return result;
        }

        Arrays.sort(candidates);
        helper(result, new ArrayList<>(), target, candidates, new boolean[candidates.length], 0);
        return result;
        
    }

    private void helper(List<List<Integer>> result, List<Integer> item, int target, int[] candidates, boolean [] visited, int index) {
        if (target == 0) {
            result.add(new ArrayList<>(item));
            return;
        }

        if (target < 0 || index == candidates.length || candidates[index] > target) {
            return;
        }

        for (int i = index; i < candidates.length; i++) {
            if (visited[i] == true) {
                continue;
            }
            if (i > 0 && candidates[i] == candidates[i-1] && !visited[i-1]) {
                continue;
            }
            visited[i] = true;
            item.add(candidates[i]);
            helper(result, item, target - candidates[i], candidates, visited, i + 1);
            visited[i] = false;
            item.remove(item.size() - 1);
        }
    }
}
```
