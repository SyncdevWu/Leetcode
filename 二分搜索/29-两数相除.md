
```
给定两个整数，被除数 dividend 和除数 divisor。将两数相除，要求不使用乘法、除法和 mod 运算符。

返回被除数 dividend 除以除数 divisor 得到的商。

整数除法的结果应当截去（truncate）其小数部分，例如：truncate(8.345) = 8 以及 truncate(-2.7335) = -2

 

示例 1:

输入: dividend = 10, divisor = 3
输出: 3
解释: 10/3 = truncate(3.33333..) = truncate(3) = 3
示例 2:

输入: dividend = 7, divisor = -3
输出: -2
解释: 7/-3 = truncate(-2.33333..) = -2
 

提示：

被除数和除数均为 32 位有符号整数。
除数不为 0。
假设我们的环境只能存储 32 位有符号整数，其数值范围是 [−231,  231 − 1]。本题中，如果除法结果溢出，则返回 231 − 1。
```

```
class Solution {
    public int divide(int dividend, int divisor) {
        
        long sign = 1;
        if ((dividend > 0 && divisor > 0) || (dividend < 0 && divisor < 0)) {
            sign = 1;
        } else {
            sign = -1;
        }
        long ldividend = dividend;
        long ldivisior = divisor;
        ldividend = ldividend < 0 ? -ldividend : ldividend;
        ldivisior = ldivisior < 0 ? -ldivisior : ldivisior;
        long count = 1;
        long result = 0;
        while (ldividend >= ldivisior) {
            ldividend -= ldivisior;
            result += count;
            ldivisior <<= 1;
            count <<= 1;
          
        }
        ldivisior >>= 1;
        count >>= 1;
        while(count > 0 && ldividend > 0 && ldivisior > 0) {
           if (ldividend >= ldivisior) {
               ldividend -= ldivisior;
               result += count;
           } else {
               count >>= 1;
               ldivisior >>= 1;
           }
           
        }
        result *= sign;
        if (sign == 1) {
            return result > Integer.MAX_VALUE ? Integer.MAX_VALUE : (int) result;
        } else {
            return result < Integer.MIN_VALUE ? Integer.MIN_VALUE : (int) result;
        }
    }
}
```
